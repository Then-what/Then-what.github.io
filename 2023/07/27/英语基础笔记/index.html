英语基础笔记

发表时间：2023-07-27 阅读： 评论：

English English

英语的基本句型

  • 可以独立完成的动作：主语 + 不及物动词

  • 有一个动作承受者：主语 + 及物动词 + 宾语 

  • 有两个动作承受着：主语 + 双及物动词 + 间接宾语 + 直接宾语

  • 只有一个动作承受着：主语 + 复杂及物动词 + 宾语 + 宾语补语

  • 非”动作”：主语 + (连)系动词 + 主语补语(表语)

句型补充

  • 定语：用于补充主语或宾语

  • 状语：用于修饰谓语动词

  • 同位语：用于把主语或宾语再说一遍

词性

  • **名词(n)**：表示人和物

  • **冠词(art)**：说明人和事物

  • **代词(pron)**：替代人和物

  • **形容词(adj)**：形容人和物

  • **数词(num)**：表示数量

  • **副词(adv)**：修饰动作或形容词

  • **介词(prep)**：表示和其它词的关系的词

  • **叹词(int)**：表示感叹

  • **连词(conj)**：连接词和句
