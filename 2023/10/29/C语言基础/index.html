C语言基础

注：双撇号是字符串，单撇号是字符常量

前言

C程序主要是由函数构成的，一个C源程序必须包含一个main函数，或main函数和若干个其它函数，因此，函数是C程序的基本单位。

一个C程序总是从main函数开始执行的，无论main函数在程序的什么位置都一样。

C语言本身是没有输入输出语句，输入输出是由库函数 scanf 和 printf 等函数来完成的。

基础

C程序的运行步骤

程序，就是一组计算机能识别和执行的指令，用高级语言编写的程序为“源程序”，但计算机只能识别并执行0和1组成的二进制的指令，所以需要“编译程序”的软件，将高级程序编译成二进制形式的“目标程序"，然后再将目标程序与系统函数库以及其它目标程序连接起来，形成”可执行程序“。

强制类型转换

格式：(类型名)(表达式)

例如：(double)a (float)(x/y) (int)(x+y)

注：如果只写成 (int)x + y，就会先将 x 强制转换为 int 整型，然后再进行运算。

补码的算法

取该数（不考虑符号）的二进制形式，这就是该数的原码。

使用原码取反，就是该数的反码。

使用反码加 1，就是该数的补码。

数据的输入输出

函数名 作用 例子
putchar() 向显示器输出一个字符 char a = ‘B’; putchar(a);
getchar() 输入一个字符 char a; a = getchar();
scanf() 输入数据 char a, int b; scanf(“%c,%d”, &a, &b)
printf() 输出数据 char a = ‘A’, int b = 100; printf(“a=%c,b=%d”, a, b)

注：sancf()函数中的例子，& 是地址运输符，&a 是指变量 a 在内存中的地址，输入的数据会一次进入这些变量的存储单元中

格式字符串

格式化字符串 作用
%d,i 输入输出时用十进制整数表示(显示)
%c 字符输入输出时所用的格式字符串
%s 用来输入输出一个字符串
%f 用来输入输出一个实数
%u 用来输入输出无符号十进制整数
%o 以八进制的形式输入输出
%x 以十六进制的形式输入输出
%e 用来输出数字的科学计数法
%g 根据变量的长度自动选择 %f 或 %e 的格式输出

注：使用 %f 输出 double 类型的时候，需要在 f 前面添加一个大写字母 L 或小写 l 修饰符，如：double a = 12.122; printf(“%lf”, a);

格式字符串的输入输出修饰符

修饰符 作用
域宽 用于指定输入输出数据所占的宽度
l 用于输入输出长整型的数据
h 用于输入输出短整型的数据
* 表示输入项在读入后不赋值给相应的变量

数学函数

注：使用数学函数的时候要在 #include <stdio.h> 的下面添加 #include <math.h>

函数名 作用
sqrt() 求平方根

数组

**数组格式：**数组类型 数组名 [数组长度] = {数组初始值}

**例子：**int arr[10] = {1, 2, 3, 4, 5, 6, 7, 8, 9, 0}

字符串处理函数

注：使用字符串函数时，需要在文件头部添加 #include <string.h>

函数名 作用
gets(字符数组) 从终端输入一个字符串到数组里
puts(字符数组) 将一个字符串输出到终端
strcat(字符数组1, 字符数组2) 合并两个字符数组，将字符数组2连接到字符数组1后面
strcpy(字符数组1, 字符串2) 将字符串2直接覆盖字符数组1的数据
strcmp(字符串1, 字符串2) 比较两个字符串，两个字符串相等返回 ”0“，1 大于 2 返回 ”1“，1 小于 2 返回 ”-1“
strlen(字符数组) 返回字符数组的长度
strlwr(字符串) 将字符串大写字母转换成小写字母
strupr(字符串) 将字符串小写字母转换成大写字母

函数

存储类别

定义：C语言中有两种存储类别，分别为静态存储方式和动态存储方式，静态存储方式会在程序运行期间系统在静态存储区划分存储空间，而在程序运行期间静态存储的数据不会被释放，动态存储方式则是在函数的调用期间在动态存储区划分存储空间，当函数运行结束动态存储方式的数据会被释放

指定变量存储类别

  1. 自动变量**(**动态变量)

    **修饰符：**auto

    **作用：**将变量使用动态存储方式存储

    **例子：**auto int a, b = 3;

  2. 静态变量

    **修饰符：**static

    **作用：**将变量使用静态存储方式存储

    例子： static int a, b;

    注：在函数中使用这个修饰符，会将数据转换成静态局部变量，也就是说当函数运行结束之后，并不会将改变量释放，会保留上一次运行的值

内部函数和外部函数

  1. 内部函数

    **修饰符：**static

    **作用：**用于将函数声明为内部函数，使其不能被其它文件使用

    **例子：**static int max(int x, int y){};

  2. 外部函数

    **修饰符：**extern

    **作用：**用于将函数声明为外部函数，使其能被其它文件使用，如果不加修饰符，默认是外部函数(extern)

    **例子：**extern int max(int x, int y){};
